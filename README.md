## Installation

1. Clone the repository:

   ```bash
   https://github.com/panderonak/background-changer.git
   ```

2. Navigate to the project directory:

   ```bash
   cd background-changer

   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

- Start the application:

  ```bash
  npm run dev
  ```
